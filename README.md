## Featured Projects

### [Review-Gate (V2)](https://github.com/LakshmanTurlapati/Review-Gate)
A sophisticated rule for the Cursor IDE that turns single AI requests into interactive, multi-turn development sessions. V2 introduces a professional UI, voice commands, and vision capabilities.
- **Features**: Integrated Popup UI for seamless conversations, Voice Commands with local Faster-Whisper, and Vision Support to upload screenshots.
- <sub>**Tech Stack**: Python, AI Agents, IDE Integration, UI/UX</sub>

---

### [t2s-cli](https://github.com/LakshmanTurlapati/t2s-cli)
A privacy-first, pip-installable CLI tool that converts natural language into SQL queries. It runs entirely on your local machine, ensuring your data never leaves your environment.
- **Features**: Hardware-Aware AI model selection, auto-detection of local databases, and fully offline functionality for security.
- <sub>**Tech Stack**: Python, CLI, SQL, NLP, Hugging Face</sub>

---

### [Parz-AI](https://github.com/LakshmanTurlapati/Parz-AI)
An all-in-one solution for creating, fine-tuning, and deploying a personal AI persona on consumer hardware using small language models (SmolLM).
- **Features**: Fine-tune a base model on personal data, convert to optimized GGUF format, and host as a local API server.
- <sub>**Tech Stack**: Python, AI, LLM Fine-Tuning, GGUF, API</sub>

---

### [FormsiQ](https://github.com/LakshmanTurlapati/FormsiQ)
An intelligent app that transforms mortgage call transcripts into completed Uniform Residential Loan Application forms (Form 1003) using AI-powered field extraction.
- **Features**: Intelligent Data Extraction with context-awareness, advanced PDF form filling, and local processing for maximum data privacy.
- <sub>**Tech Stack**: Python, Django, Angular, NLP, PDF Automation</sub>
